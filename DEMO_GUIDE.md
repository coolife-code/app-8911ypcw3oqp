# 碎念小栈 - 功能演示指南

## 🎮 完整功能演示流程

### 场景1：首次使用

#### 步骤1：打开网站
```
✨ 看到像素风格的界面
✨ 标题"碎念小栈"带有星星闪烁效果（午夜模式）
✨ 像素垃圾桶自动抖动
✨ 背景有像素猫动画
```

```

   "今天加班到很晚，感觉身体被掏空"
```

```
纸条3 - 微小说 📖

纸条4 - 哲理名言 💎
  "当你觉得累的时候，说明你还在向上爬。躺平的人从不喊累。"

💡 点击任意纸条可以翻面查看详细内容
```

## 💡 使用技巧

### 技巧1：寻找最佳回应
```
1. 输入心情
2. 查看第一次生成的结果
3. 如果不满意，点击"用这张再碎"
4. 重复步骤3，直到找到最喜欢的回应
5. 建议尝试3-5次，选择最有趣的
```

### 技巧2：比较不同风格
```
1. 输入相同的心情
2. 多次重新碎纸
3. 比较不同版本的回应
4. 观察AI生成的多样性
5. 选择最符合心情的风格
```

### 技巧3：探索AI创意
```
1. 输入简短的心情（如"累了"）
2. 重新碎纸多次
3. 观察AI如何扩展简短的输入
4. 发现意想不到的有趣回应
```

### 技巧4：长文本处理
```
1. 输入详细的心情描述（100-200字）
2. 观察AI如何提取核心情绪
3. 重新碎纸看不同的理解角度
4. 比较哪个版本最准确
```

## 🐛 常见问题演示

### 问题1：AI返回格式错误
```
❌ 错误提示："AI小精灵打瞌睡了，请再试一次 😴"

✅ 解决方法：
   1. 点击重试按钮
   2. 系统自动修复格式问题
   3. 大多数情况下第二次就能成功
```

### 问题2：网络连接问题
```
❌ 错误提示："网络连接不稳定，请检查网络后重试 📡"

✅ 解决方法：
   1. 检查网络连接
   2. 重新点击"碎一下"
   3. 或点击"用这张再碎"重试
```

### 问题3：生成时间过长
```
⏳ 正常情况：3-10秒
⚠️ 超过15秒可能有问题

✅ 解决方法：
   1. 等待完成
   2. 如果超过30秒，刷新页面
   3. 重新尝试
```
