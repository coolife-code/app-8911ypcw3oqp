# 移动端响应式优化说明

## 📱 优化概述

针对移动端进行了全面的响应式优化，确保在小屏幕设备（手机、平板）上也能获得良好的体验。所有元素都根据屏幕尺寸自动调整大小和位置。

## 🎯 优化目标

- ✅ 卡片大小适合小屏幕
- ✅ 文字清晰可读
- ✅ 按钮易于点击
- ✅ 界面不拥挤
- ✅ 拖拽操作流畅

## 🎨 响应式断点

使用Tailwind CSS的`xl`断点（≥1280px）：

```css
/* 移动端（默认）：<1280px */
.card {
  width: 10rem;  /* w-40 */
  height: 14rem; /* h-56 */
}

/* 桌面端：≥1280px */
@media (min-width: 1280px) {
  .card {
    width: 16rem;  /* xl:w-64 */
    height: 24rem; /* xl:h-96 */
  }
}
```

## 🔧 技术实现

### 响应式类名模式

```tsx
// 卡片尺寸
className="w-40 h-56 xl:w-64 xl:h-96"
// 移动端：40×56
// 桌面端：64×96

// 文字大小
className="text-4xl xl:text-6xl"
// 移动端：text-4xl
// 桌面端：text-6xl

// 内边距
className="p-3 xl:p-6"
// 移动端：p-3
// 桌面端：p-6

// 位置
className="bottom-4 right-4 xl:bottom-8 xl:right-8"
// 移动端：bottom-4 right-4
// 桌面端：bottom-8 right-8
```

### 代码示例

**PaperStrip.tsx**：
```tsx
<div className={cn(
  'flip-card-inner w-40 h-56 xl:w-64 xl:h-96',
  isFlipped && 'flipped'
)}>
  <div className={cn(
    'w-full h-full pixel-border border-foreground rounded-lg',
    'p-3 xl:p-6',  // 响应式内边距
    'flex flex-col items-center justify-center',
    'gap-2 xl:gap-4',  // 响应式间距
    'bg-card shadow-xl'
  )}>
    <div className="text-4xl xl:text-6xl">{config.emoji}</div>
    <h3 className="text-[10px] xl:text-sm font-bold pixel-text text-center">
      {config.title}
    </h3>
    <p className="text-[8px] xl:text-xs text-muted-foreground text-center">
      点击翻面 / 拖到垃圾桶
    </p>
  </div>
</div>
```

**CardStack.tsx**：
```tsx
{/* 垃圾桶 - 响应式尺寸和位置 */}
<div className={cn(
  'fixed bottom-4 right-4 w-20 h-20',  // 移动端
  'xl:bottom-8 xl:right-8 xl:w-40 xl:h-40',  // 桌面端
  'pixel-border border-foreground rounded-lg',
  'flex flex-col items-center justify-center gap-1 xl:gap-2',
  'transition-all duration-300'
)}>
  <div className="text-3xl xl:text-6xl">🗑️</div>
  <p className="text-[8px] xl:text-xs pixel-text text-center">
    拖到这里
  </p>
</div>

{/* 按钮 - 响应式尺寸和位置 */}
<div className="fixed bottom-4 left-4 xl:bottom-8 xl:left-8">
  <button className="pixel-border border-foreground bg-card px-3 py-2 xl:px-6 xl:py-3 rounded-lg font-bold hover:bg-accent transition-colors text-[10px] xl:text-xs">
    输入新内容 ✨
  </button>
</div>
```

## 📐 设计原则

### 1. 移动优先
- 默认样式针对移动端设计
- 使用`xl:`前缀添加桌面端样式
- 确保移动端体验不打折扣

### 2. 比例缩放
- 移动端元素按比例缩小
- 保持视觉层次和间距比例
- 文字大小适中，易于阅读

### 3. 触摸友好
- 按钮和垃圾桶大小适合手指点击
- 卡片间距合理，避免误触
- 拖拽操作流畅自然

### 4. 内容优先
- 重要内容优先显示
- 提示信息简洁明了
- 避免信息过载
